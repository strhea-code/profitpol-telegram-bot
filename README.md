# Telegram бот для учета выполненных работ

Telegram-бот для сотрудников компании.  
Позволяет вводить данные о выполненных работах и автоматически записывает их в Google Sheets.

## Функции

Бот позволяет:

- проверять доступ по Telegram ID
- выбирать объект
- выбирать дату выполнения работ
- выбирать наименование работ
- вводить объем выполненных работ
- записывать данные в Google Sheets
- формировать отчет за день для администратора

## Команды

`/start` — начать ввод данных  
`/myid` — показать свой Telegram ID  
`/today_report` — отчет по всем работам за сегодня (только для администратора)

## Технологии

- Python
- pyTelegramBotAPI
- Google Sheets API
- gspread
- Railway (хостинг)

## Структура таблицы

Лист **Начисления**:

| Колонка | Значение |
|-------|-------|
| A | ФИО |
| B | Объект |
| D | Дата |
| I | Наименование работ |
| K | Объем |

Лист **Доступ**:

| Колонка | Значение |
|-------|-------|
| A | ФИО |
| B | Telegram ID |
| C | Роль (admin для администратора) |

## Переменные окружения

В Railway используются следующие переменные:


TOKEN

SPREADSHEET_NAME

GOOGLE_CREDS_JSON


## Запуск

Бот запускается на сервере Railway.

Команда запуска указана в файле Procfile:

worker: python bot.py
